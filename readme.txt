Plugin just adds custom field “buy_price” to WooCommerce products. No more, no less.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-field-buy-price` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
